# solace-architect-discovery

**SAM agent plugin** — Structured discovery interview + intake parsing + reference-architecture pattern matching.

**Status:** Phase 0 scaffold — no working code yet. See [v2spec §4.2](../../documents/v2spec.md).

## Install (development)

```bash
pip install -e ../../solace-architect-core/
pip install -e .
```

## Install (end users, after community-repo PR merges)

```bash
sam plugin add solace-architect-discovery --plugin solace-architect-discovery
```
