# solace-architect-reviewer-architect

**SAM agent plugin** — Architecture-perspective reviewer — trade-offs, component choices, simpler alternatives, reference-architecture alignment.

**Status:** Phase 0 scaffold — no working code yet. See [v2spec §4.4](../../documents/v2spec.md).

## Install (development)

```bash
pip install -e ../../solace-architect-core/
pip install -e .
```

## Install (end users, after community-repo PR merges)

```bash
sam plugin add solace-architect-reviewer-architect --plugin solace-architect-reviewer-architect
```
